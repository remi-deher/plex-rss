Seer → Paramètres → Général → Clé API
Rôles
Les nouvelles demandes sont transmises à Seer plutôt que directement à Sonarr/Radarr. Sonarr/Radarr restent utilisés pour le suivi de statut et les profils qualité.
Si l'envoi via Seer échoue (timeout, erreur), la demande est automatiquement retentée directement sur l'instance Sonarr/Radarr correspondante.
Actions
Synchronise les comptes Seer avec les utilisateurs Plex et met à jour les badges Hybride / Seer.
