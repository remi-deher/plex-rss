Webhooks — Notifications de disponibilité

Configurez ces URLs dans Sonarr / Radarr → Paramètres → Connexions → Webhook pour être notifié dès qu'un contenu est téléchargé et importé. Sélectionnez l'événement OnDownload / OnImport.

Sonarr → Paramètres → Connexions → Webhook · Événement : OnDownload / OnImport
Radarr → Paramètres → Connexions → Webhook · Événement : OnDownload / OnImport

Plex → Paramètres → Webhooks → Ajouter un webhook · Événements traités : library.new (nouveau contenu) et media.scrobble (regardé en entier). Permet une détection de disponibilité instantanée sans attendre la vérification périodique.
La correspondance se fait en priorité par TMDB/TVDB/IMDB ID, puis par titre si les IDs sont absents.
