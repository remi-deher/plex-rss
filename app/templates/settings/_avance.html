Webhooks — Notifications de disponibilité

Collez ces URLs (secret inclus) dans Sonarr / Radarr → Paramètres → Connexions → Webhook (événement On Import / On Grab) et dans Plex → Paramètres → Webhooks. Les liens sont générés à partir de l'adresse de votre navigateur.

Généré automatiquement au premier lancement. Le régénérer invalide les webhooks déjà configurés avec l'ancien secret (il faudra recoller les nouvelles URLs).

Radarr → Connexions → Webhook · Événement : On Import / On Grab

Plex → Paramètres → Webhooks · Événements traités : library.new et media.scrobble. Détection de disponibilité instantanée sans attendre la vérification périodique.
